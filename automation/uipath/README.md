# Intelligent Settlement Automation System

This project demonstrates an end-to-end automation solution combining:

- Python-based anomaly detection (Decision Layer)
- UiPath RPA bot (Execution Layer)
- Business Analysis artefacts (UAT, ROI, Process Design)

## Architecture

Python → Detect anomalies → UiPath → Process & Alert

## Features

- Automated mismatch detection
- RPA-based reconciliation workflow
- Exception handling and logging
- Business process documentation

## Tech Stack

- Python (Pandas)
- UiPath (RPA)
- Excel Automation
